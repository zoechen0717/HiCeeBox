# Examples

Example configs: `example_config.yaml`, `minimal_example.yaml`, `python_api_example.py`. The Jupyter notebooks `HiCeeBox_Tutorial.ipynb` and `HiCeeBox_Demo.ipynb` show the Python API.

Configs expect data in a `data/` subfolder (e.g. `example.hic`, `*.bw`, `peaks.bed`, `loops.bedpe`, `genes.gtf`). Point paths to your own files.
